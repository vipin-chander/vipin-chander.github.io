Trekmate

OVERVIEW

TrekMate is an app which helps you to trek on your own, I.e without any help of any guide/person. Just need your phone with GPS inbuilt and with enough battery ofcourse.
It helps you to plan your trek accordingly, provided better weather and feasible condition. It tells you which season is better for doing the trek.

ROLE

User Experience, User Research, Prototyping, Interaction, User Interface Design

TIMELINE

October 2019 - November 2019

GOAL

  1. For a trekker/hiker the main challenge is always how to continue trekking when he/she donâ€™t know the trail/route, this is where this app helps.
  2. So One and only goal of this app is to help a trekker/hiker to do the trek on his own without any help of a guide/person with proper planning.

CHALLENGE

  1. How to accumulate or collect detailed data of the treks.
  2. How to show this info in a very easy and communicative way to an unprofessional trekker or a newbie to the treks.

ASSUMPTION

  1. Have all the data of treks like map, coordinates (start point - end point - basecamps, fresh water source, nearby village etc) which can collected by various organisations.
  2. Treks which are listed in the app must not be unexplored treks i.e a trekker can only trek when the same trek has been trekked down before.

SOLUTION

Idea of landmarks came into my mind, and yes they are very helpful in daily life too, then why not in trekking. What can be done is to provide the trail/route direction along with the landmarks. A map with every details a trekker needs like basecamp location, water source location (if any), nearby village direction (if any), etc. Along with this any landslide prone region information or something like that. User will be prompt via notification about the timing of the trek, I.e when to start or not, when to move back or not, what is best time to summit or by which time he must turn back to basecamps.
The key to successful trek/summit is to follow the proper timings.

KEY FEATURES

  1. Provides basecamp, water source, any nearby village locations.
  2. Provides landmarks which helps in trekking along with the map. A user can be asked whether he crossed the specific landmark or not, which will confirm that he/she is on right trail or not.
