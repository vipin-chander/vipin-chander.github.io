Mobikwik Customer Support

OVERVIEW

MobiKwik is digital financial services platform, which helps user to recharge their bill, pay online on merchants like Myntra, Flipkart, Croma, IRCTC and you can do much more.As being a B2C product, we also had to take care of day to day transaction related problems of the users, we had to minimize them and solve them smartly and efficiently.

This project is live and you can check it here

ROLE

User Experience, Web Development, Interaction, Prototyping

TIMELINE

March 2018 - June 2018

GOAL

  1. To solve the transaction related problem in a very easy and effecient way
  2. To Lower the number of tickets raised per day
  3. Make it automatice ticketing system so as to decrease work load of operational associates

CHALLENGE

Main challenge was to make it like a automatic system which solves the problem of user without taking help of any opertional associate and AI.

KEY FEATURES

  1. Automatically solves the problem of user
  2. Lower the number tickets raised per day

ANALYSIS

  1. Analysed all the possible cases of failed transaction.
  2. And according to the problem, made set of questions based on possible cases.
  3. Took internal feedback i.e. from the employees of the company
  4. Re-evaluated and then Redesigned
