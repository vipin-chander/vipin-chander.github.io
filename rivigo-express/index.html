Transfer Money for SME

Project Type : Web App
Organisation : NIUM
No. of Team Member(s) : 1

Project Overview

RIVIGO Express is a one-stop solution for all your shipping related needs. With over 29,000+ pin codes serviced through 4,000+ self-owned trucks. RIVIGO is a new age technology enabled company ensuring fastest delivery times (30% lower than competition) and record lowest damages (5x lower than competition).

Role

User Experience, Interaction, Prototyping & User Interface Design

Timeline

October 2019 - November 2019

Objectives

  1. Drive adoption by creating ease of discovery for newly launched features
  2. Decrease manual work of the agents on booking
  3. Adopt modern UI paradigms for a more user friendly interface for Express app

Solution Proposed

1st Motive was to make the app single handed and user friendly. Why user friendly? Because

  1. Most users prefer single handed ( ~85% based on a study by UX matters )
  2. With increasing screen sizes, more & more Apps are adopting one handed user experience.

2nd motive was discoverability of items and new features

  1. Generally home page is start of user journey. So we had to put features on home page instead of hamburger menu. (Old home page screenshot)
  2. So we redefined the home page. On this page book feature is a single click feature as location gets picked up automatically so in order to get new and simple and clearer UI, we had to change it anyhow. (new home page screenshot).

Analysis

  1. POD section moved to order details section of the corresponding order instead of separate part, in old flow one has to enter CN number to get POD, which is very hard to remember (Old menu and new order details screens)
