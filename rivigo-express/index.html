Rivigo Express

OVERVIEW

RIVIGO Express is a one-stop solution for all your shipping related needs. With over 29,000+ pin codes serviced through 4,000+ self-owned trucks.
RIVIGO is a new age technology enabled company ensuring fastest delivery times (30% lower than competition) and record lowest damages (5x lower than competition).

ROLE

User Experience, Interaction, Prototyping & User Interface Design

TIMELINE

October 2019 - November 2019

GOAL

  1. Drive adoption by creating ease of discovery for newly launched features
  2. Decrease manual work of the agents on booking
  3. Adopt modern UI paradigms for a more user friendly interface for Express app

CHALLENGE

Main challenge was to design single handed and more prominent UI in terms of discoverability i.e. had to work hard on home screen fo the app. So one thing we were sure about to change home screen entirely.

SOLUTION

1st Motive was to make the app single handed and user friendly. Why user friendly? Because

  1. Most users prefer single handed ( ~85% based on a study by UX matters )
  2. With increasing screen sizes, more & more Apps are adopting one handed user experience.
2nd motive was discoverability of items and new features
  1. Generally home page is start of user journey. So we had to put features on home page instead of hamburger menu. (Old home page screenshot)
  2. So we redefined the home page (new home page screenshot)

KEY FEATURES

  1. Segregation of overall APP flow into key sections
  2. Lack of a left hamburger selection

ANALYSIS

  1. Currently, our home page only caters to “book cargo feature” which also mis communicates. According to the data we got 50% of the booked requests were fake because of the very wrong design which doesn’t tell what is going on or what is gonna happen next. So in this case user accidentally place the request. (Old and new home screen respectively)
  2. Book feature is a single click feature as location gets picked up automatically so in order to get new and simple and clearer UI, we had to change it anyhow. (Old and new home screen respectively)
  3. Call me feature is present on every screen which can be possibly replaced with providing option to make a call + call me or separate help section (My account screen)
  4. POD section moved to order details section of the corresponding order instead of separate part, in old flow one has to enter CN number to get POD, which is very hard to remember (Old menu and new order details screens)
